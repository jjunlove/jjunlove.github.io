# 이미지 에셋 체크리스트 (자동 생성 — tools/list-assets.mjs)

파일을 아래 경로/이름으로 `img/` 하위에 넣으면 상세 페이지가 자동으로 텍스트 폴백 대신 이미지를 씁니다.
코드 수정 불필요. `[x]` = 이미 존재, `[ ]` = 없음(폴백 사용 중).

- **카운터**(기체 실루엣): 권장 크기 ~52×34px 내외, 투명 PNG. 파일명 = 기체명 슬러그.
  같은 실루엣을 여러 기체에 쓰려면 md 유닛 줄에 `| counter: 공유키` 를 넣어 파일명을 통일하세요.
- **배경 사진**: 시나리오 id.jpg. 가로 240px 이상 권장.
- **국가 마크**: 국가 키.png(원형 아니어도 됨). ~30×30px.
